Introdução

Este sistema de navegação assistiva foi criado para melhorar a acessibilidade e facilitar o uso de dispositivos móveis para pessoas com necessidades especiais.

Como Funciona

O sistema usa tecnologias como GPS, sensores de movimento e interação por voz para guiar o usuário de forma segura e eficiente.

Nosso Projeto com Arduino

O projeto de navegação assistiva com Arduino foi desenvolvido com o objetivo de criar uma solução acessível e eficaz para pessoas com deficiências visuais e motoras. A principal ideia é usar o Arduino para coletar informações de sensores, processá-las e fornecer feedback para o usuário em tempo real.

Componentes Utilizados:

  • Arduino Uno
  • Sensores de proximidade (ultrassônicos)
  • Sensor de distância LIDAR
  • Vibradores de feedback
  • Modulo Bluetooth
  • Fonte de alimentação

Como Funciona o Sistema:

O sistema utiliza sensores de proximidade para detectar obstáculos ao redor do usuário. Os sensores ultrassônicos mapeiam a distância dos objetos e, com base nesse mapeamento, o Arduino processa as informações e envia comandos para os vibradores de feedback. O sistema também é capaz de interagir com o usuário através de um aplicativo móvel via Bluetooth, permitindo configurações e alertas personalizados.

Aplicações do Projeto:

  • Navegação assistiva para deficientes visuais em ambientes internos.
  • Auxílio na locomoção de pessoas com mobilidade reduzida.
  • Desenvolvimento de dispositivos móveis que interagem com a tecnologia Arduino para melhor experiência.

Este projeto visa proporcionar mais independência e segurança para as pessoas com necessidades especiais, permitindo que elas naveguem com maior facilidade em ambientes complexos, como shoppings, aeroportos, hospitais e outros lugares públicos.
